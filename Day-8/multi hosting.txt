Task: Can you explore if we can do multi-site hosting using Azure Load Balancers and App Gateways

Ans:

1. Azure Load Balancer (Layer 4)

Works at TCP/UDP level
Routes based on:
IP address
Port

Cannot understand:

URL paths
Hostname

2. Azure Application Gateway (Layer 7)

Works at HTTP/HTTPS level
Can route based on:
Hostname (domain)
Path (/api, /images)
Supports multi-site hosting

Can we do Multi-site hosting?

Using Load Balancer ONLY ->NO

Using Application Gateway Feature ->YES
